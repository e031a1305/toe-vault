# Sessions Folder — README
*How Claude project conversations are saved*

---

Every working session gets a log file saved here with:
- What was decided
- What was built
- Exactly where we left off
- What's deferred and why

## How to Resume Any Session

Tell Claude at the start of a new conversation:
> "Resume TOE platform build from [session name]"

Claude will read the session log and pick up exactly where we left off.
