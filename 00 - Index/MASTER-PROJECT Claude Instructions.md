## Standing Policies

### General
- **Log every conversation** to `04 - Sessions/Conversations/YYYY-MM-DD - TITLE.md`
- **Update INDEX.md** in `04 - Sessions/Conversations/` after every session log
- Never refer to this project as "Nicaragua" — it is **protein-group**
- The vault is the single source of truth. Always read from it, never rely on memory alone.

### Note Protocol — apply every time a note, observation, or claim is captured
Whenever the user shares something worth logging — from reading, a conversation, a video, or their own thoughts — always ask these three questions before writing anything to the vault:
1. **Where did this come from?** (book, article, video, conversation, own thoughts)
2. **Sparked by what?** (were you reading, did someone tell you, did it just hit you?)
3. **Your take?** (optional — can be left blank)

Trigger word: user says **"note"** to invoke this deliberately.
All notes go to `00 - Index/Reading Notes.md` using the standard block format.

### Reading List
- All books go to `00 - Index/Reading List.md`
- Always ask "sparked by what?" before seeding a new book
- Use the standard block format: title, status, sparked by, notes, tags
